Introduction

Python is a high-level programming language suitable for beginners thanks to its intuitive syntax, huge community, and vibrant ecosystem. Technically speaking, Python is an interpreted language that does not have an intermediate compilation phase like a compiled language, for example C or Java.

And like many interpreted languages, it is typed dynamically. This means that you do not have to indicate the types of variables you use and the variables are not bound to a specific type. Python is a programming language widely used in various areas, including:

  • Web Development: It is popular for back-end web development for its simplicity, scalability, and robust frameworks like Django and Flask.
  • Data Science: Python's powerful libraries for numerical analysis, such as NumPy, Pandas, and scikit-learn, make it an essential tool for data scientists and analysts.
  • Machine Learning: Python is the leading language in the field of machine learning thanks to its ease of use and frameworks.
  • Software development: It is used to create a wide range of applications, from desktop software to productivity tools.
  • Automation: Python's simplicity and flexibility make it ideal for automating repetitive tasks and streamlining workflows.

Indentation

Indentation refers to the spaces at the beginning of a code line. It defines the structure of your code and determines which lines belong together as a block.

Everything indented belongs to a block, such as a control statement or conditional block, or a function or class body. An incorrect indentation will lead to syntax errors and your code won't run.

Indentation’s rules:

  • Use 4 spaces for each indentation level. This is the recommended and widely adopted convention.
  • All statements within a block must have the same indentation.
  • Consistency is key! Maintain the same indentation throughout your code for readability.
What is a data type?

A data type refers to the classification of values ​​that can be stored in a variable. Each type of data has specific characteristics and properties that make it suitable for certain uses and operations.

Data types are also used to determine how values ​​are stored in computer memory and how the values ​​can be manipulated and operated on. For example, integers are stored as sequences of bits in memory, and arithmetic operations such as addition, subtraction, multiplication, and division can be performed.

Data Types

The types of data play are crucial when building robust and efficient applications. As one of the most popular and versatile programming languages, Python offers a wide range of built-in data types that make it easy to manipulate and store information. There are many data types in Python, some of them are -

Integer (int)

Data of type int (short for "integer"), in Python, are those that represent integers. These are a type of numbers that do not have a decimal part, they are complete without fractions.

1, 2, 3, 4, 5

Float

Float data (short for “floating point”), in Python, are those that represent numbers with decimals.

1.23 3.14 45.5 -12.34

Complex Numbers (complex)

Those that represent numbers that have a real part and an imaginary part. In mathematics, complex numbers are defined as the sum of a real part and an imaginary part multiplied by the imaginary unit (denoted as "i" or "j").

NOTE - Complex data is represented by writing a real number followed by a "j" to indicate the imaginary part.

3 + 2j -7.5 - 1j 2j

NOTE - Instead of "i", Python uses "j", to denote the imaginary unit.

Text Types (str)

String data (short for "character string" or "string"), in Python, is data that represents a sequence of characters (letters, numbers, symbols, etc.). They are represented in single quotes('')or double quotes("")

"Hello World" ’Python is great!' "1234"

Booleans (bool)

Data type bool (short for "boolean"), are those that represent truth values. In other words, they are true or false values. In Python, bool data is represented as True or False (with the first letter capitalized).

They are primarily used in logical and control-flow expressions, such asifandwhilestatements.

Sequence types (list)

Lists are those that represent an ordered collection of elements. In other words, lists are a data structure that allows multiple values ​​to be stored in a single object and accessed by their position in the list. List-type data is represented by brackets «[ ]» that contain the list elements separated by commas.

[1, 2, 3, 4, 5] ['a', 'b', 'c'] [1, 'a', True, [2, 3, 4]]

Tuples

Tuples are a data structure that allows multiple values ​​to be stored in a single object and accessed by their position in the tuple, but once the tuple is created, the elements cannot be modified. Tuple-type data is represented by parentheses( )that contain the elements of the tuple separated by commas.

(1, 2, 3, 4, 5) ('a', 'b', 'c') (1, 'a', True, (2, 3, 4))
